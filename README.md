# WaveProt

WaveProt app is used to patch Android games targeting Quest headsets. Functionality will be expanded in the future.

Powered by [Compose Multiplatform](https://www.jetbrains.com/compose-multiplatform/) and [ARSCLib](https://github.com/REAndroid/ARSCLib). [Builds](https://github.com/WaveProt/app/releases) are available for Windows and Android.

# Issues

Please, report issues related only to the app itself. If you have any issues regarding game compatibility, see [WaveProt/tracker](https://github.com/WaveProt/tracker).
